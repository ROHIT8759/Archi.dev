# Archi.dev — Project Roadmap

**Project:** Archi.dev (Visual Backend Architecture Platform)
**Milestone:** UI/UX Overhaul — "Ultra-Premium Developer SaaS" Identity
**Created:** 2026-03-25

---

## Overview

Overhaul the end-to-end user interface and experience to establish an "Ultra-Premium Developer SaaS" identity matching the visual fidelity of Vercel, Linear, and Stripe. The "Cyber-Glass" aesthetic governs all surfaces: deep layered morphism, neon accents, and cinematic micro-interactions.

---

## Phase 1: UI/UX Overhaul

**Goal:** Implement the full Cyber-Glass design system across all pages and components — Landing, Auth, Dashboard, Studio Canvas, Docs, and Settings — as specified in the UI Design Contract (01-UI-SPEC.md).

**Requirements**: REQ-001, REQ-002, REQ-003, REQ-004, REQ-005, REQ-006, REQ-007, REQ-008, REQ-009, REQ-010, REQ-011, REQ-012, REQ-013, REQ-014, REQ-015

**Deliverables:**
- Global design system tokens (CSS variables, typography, spacing, colors, shadows)
- Landing page cinematic experience (parallax hero, bento spotlight, sticky scroll, magnetic CTA)
- Authentication pages split-screen layout
- Dashboard mission-control layout (stats row, project cards, empty state templates)
- Studio canvas core UX (custom nodes, animated edges, command palette, floating context menu, AI copilot)
- Auto-generated docs three-pane layout
- Settings & billing credit ledger and API key management
- 15 new components + 7 upgraded components per component inventory

**Success Metrics:**
- All pages render with Cyber-Glass aesthetic
- Command palette (Cmd+K) functional for node placement
- 60fps animations on all interactive elements
- WCAG AA contrast compliance
